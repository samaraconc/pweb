Configurações no GitHub

(Para criar conta e repositorio no github) 1: Buscar no navegador por "GitHub"->Sign up->; 2: Adicionar email, criar senha e alguns dados necessarios; 3: Criar um repositorio em; "Create a new repository"; 4: Nomear o repositorio em; "Repository name"; 5: Escolher se esse repositorio será publico ou privado;

(Para entrar na conta e repositorio do GitHub)

1: Buscar no navegador por "GitHub"->Sign in->; 2: Adicionar email e senha; 3: Acessar o repositorio que fica localizado no canto esquerdo da tela;

Configurações no Eclipse:

(Para que os projetos do repositorio sejam salvas na pasta ) 1:Acessar a barra superior de feramentas -> Window -> Preferences 2:Procurar por "Git"; 3:Browse->Pasta p/ salva o repositorio do projeto->Aply and close;

(Para que a barra de estagios do github esteja visivel)

:Acessar a barra superior de feramentas -> Window -> Show view -> Other; 2:Procurar por "Git" -> selecionar "git staging" -> open;

(Para que os repositorios do github seja visualizaveis )

1:Acessar a barra superior de feramentas -> Window -> Show view ->Other; 2:Procurar por "Git" -> selecionar "git repositories" -> open;

(Para vincular o repositorio ao eclipse)

1: Clicar com o botão esquerdo do mouse no projeto; 2:Acessar "Team"-> Share Project; 3: Clicar em "Create", adicionar o nome do repositorio-> Finish;

(Para commitar o projeto)

1: Clicar com o botão esquerdo do mouse no projeto; 2: Na barra de estagios da tela em "Commit Mensage", adicionar o nome da pasta; 3: Clicar em "Commit" para que a pasta fique salvo só na pasta local; 4:1 Clicar em "Commit and Push" para que fique salvo na pasta local e no gitHub; 4:2 Copiar o endereço do projeto do github e colar na "URI"-> Preview -> Push, e assim foi enviado para o gitHub.
